# 多Agent协作自动化科研助手 - 开发任务清单

## 项目概述
输入一个研究问题，多个Agent分工协作，自动完成「文献检索 → 内容总结 → 对比分析 → 生成报告」全流程。

---

## Agent分工设计

| Agent | 职责 |
|-------|------|
| Planner | 拆解任务，分配给其他Agent |
| Searcher | 调用arXiv/语义搜索API检索论文 |
| Reader | 精读每篇论文，提取核心贡献 |
| Analyst | 横向对比多篇论文，找异同 |
| Writer | 汇总所有结果，生成结构化报告 |

---

## 技术栈
- 框架：LangGraph（节点即Agent，边即流程）
- 模型：GPT-4o 或 Claude API
- 工具：arXiv API、PDF解析（pypdf）
- 展示：Gradio Web界面

流程：
1. Planner 拆解为3个子任务
2. Searcher 检索到10篇相关论文
3. Reader 并行总结每篇论文
4. Analyst 对比各方法的优劣
5. Writer 输出一份结构化综述报告
